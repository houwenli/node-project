### 部署
    下载安装mysql，建代码仓库，拉取远程代码
    git clone https://github.com/houwenli/node-project.git
    cd myapp
    npm install
    cd www/userManage
    npm install
    
    前端部署
        cd myapp
        npm run dev
    后台部署
        cd www/userManage
        npm run dev
